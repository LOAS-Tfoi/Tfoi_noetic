## md_develop 브랜치 사용
1. 브랜치 다운로드
```bash
git clone -b md_develop https://github.com/LOAS-Tfoi/Tfoi_noetic.git
```

2. fw main을 이미 다운로드 받은경우
```bash
git fetch
git checkout md_develop
```

2-1. 브랜치가 제대로 체크아웃되었는지 확인
```bash
git status
```
